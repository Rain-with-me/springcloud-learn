# spring-boot-dubbo-demo

#### 项目介绍:
Springboot+Dubbo+Nacos 注解方式实现微服务调用

SpringBoot版本:2.2.0

Dubbo版本:2.7.0

Nacos版本:1.1.4

JDK版本:1.8



```apl
Dubbo的2.7.7版本中已经标注了Reference和Service注解为@Deprecated弃用了。改用DubboReference和DubboService这两个注解了
```
